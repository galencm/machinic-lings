# machinic-lings

_machinic DSLs and cli tools_

## Installation

Pip:

```
pip3 install git+https://github.com/galencm/machinic-lings --user
```

Develop while using pip:

```
git clone https://github.com/galencm/machinic-lings
cd machinic-lings/
pip3 install --editable ./ --user
```
